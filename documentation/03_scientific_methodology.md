# 03. Научно-инженерная методология — Industrial Vibration Analyzer

## Назначение документа

Этот документ описывает физические явления, лежащие в основе приложения, и математические методы, которые оно использует. Цель документа — дать разработчику и пользователю понимание того, почему применяются именно эти формулы и методы, и что означают получаемые результаты.

---

## Физическая основа задачи

### Потоково-индуцированные вибрации

Когда поток жидкости или газа обтекает цилиндрическое тело, за телом возникает нестационарная область течения. При определённых условиях от тела периодически отрываются вихри, которые уносятся потоком поочерёдно с двух сторон. Это явление называется вихревой дорожкой Кармана. Образование каждого вихря сопровождается изменением давления на поверхности тела, что создаёт периодическую поперечную силу.

Если частота этой силы совпадает или близка к собственной частоте колебаний конструктивного элемента (трубы, перегородки, консольного стержня), возникает явление вынужденного резонанса. В особом случае, когда конструкция начинает влиять на формирование вихрей, частота срыва синхронизируется с собственной частотой конструкции в широком диапазоне скоростей — это явление называется захватом частоты (lock-in). В режиме lock-in амплитуда колебаний резко возрастает и может привести к усталостному разрушению материала за относительно короткое время.

### Применимость к теплообменному оборудованию

В трубных пучках теплообменников, U-образных трубах и межтрубных перегородках реализуются именно эти условия: поперечный поток теплоносителя, цилиндрические элементы с известной геометрией, широкий диапазон рабочих скоростей. Поэтому оценка риска потоково-индуцированных вибраций является стандартной частью проектировочного расчёта теплообменного оборудования.

## Безразмерные критерии течения

### Число Рейнольдса

Число Рейнольдса характеризует соотношение инерционных и вязких сил в потоке и определяет режим течения — ламинарный или турбулентный.

```
Re = (V · D) / ν
```

Где:
- `V` — средняя скорость набегающего потока, м/с
- `D` — наружный диаметр цилиндра, м
- `ν` — кинематическая вязкость жидкости, м²/с

При Re < 2300 течение ламинарное. При Re > 4000 течение турбулентное. Для типичных условий в теплообменниках Re находится в диапазоне 10³–10⁵, что соответствует развитому турбулентному режиму с периодическим срывом вихрей.

### Число Струхаля

Число Струхаля связывает частоту срыва вихрей с геометрией тела и скоростью потока.

```
St = (fs · D) / V
```

Где:
- `fs` — частота срыва вихрей, Гц
- `D` — диаметр цилиндра, м
- `V` — скорость потока, м/с

Для одиночного цилиндра в диапазоне Re = 10³–10⁵ число Струхаля приблизительно постоянно и равно 0,20–0,21. Для тандема двух цилиндров значение зависит от расстояния между ними и может отличаться. В приложении используется экспериментально подтверждённое значение из диссертационного исследования.

### Расчётная частота срыва вихрей

Из определения числа Струхаля следует формула для расчёта ожидаемой частоты срыва вихрей:

```
fs = (St · V) / D
```

Эта частота является основным ориентиром при анализе спектра вибраций: если в спектре обнаружен пик вблизи `fs`, он классифицируется как вызванный вихревым срывом.

### Приведённая скорость

Приведённая скорость характеризует близость рабочего режима к условиям возникновения lock-in.

```
Vr = V / (fn · D)
```

Где:
- `fn` — собственная частота конструктивного элемента, Гц

При Vr в диапазоне 4–8 вероятность возникновения lock-in наиболее высока для одиночного цилиндра. Точные границы зависят от демпфирования конструкции и соотношения масс.

## Методы обработки сигналов

### Устранение постоянной составляющей

Реальный сигнал с датчика всегда содержит постоянную составляющую, связанную с начальным смещением, дрейфом нуля или гравитационной компонентой. Для спектрального анализа постоянная составляющая не несёт информации и должна быть удалена.

```
x_centered(t) = x(t) - mean(x)
```

### Обнаружение и удаление выбросов

Выбросы — это единичные значения, значительно отклоняющиеся от общей тенденции сигнала. Они могут возникать из-за электромагнитных помех, дребезга соединений или ошибок записи. Выбросы искажают как временное представление, так и спектральный анализ.

Метод скользящего медианного порога: в каждой точке сигнала вычисляется медиана в окне ± N отсчётов. Если значение отклоняется от медианы более чем на k стандартных отклонений, оно считается выбросом и заменяется линейной интерполяцией между соседними незатронутыми значениями.

По умолчанию: ширина окна — 21 отсчёт, порог k = 4.

### Полосовой фильтр Баттерворта

Фильтр Баттерворта выбран потому, что он обеспечивает максимально плоскую амплитудно-частотную характеристику в полосе пропускания — без пульсаций, неизбежных в фильтрах Чебышёва. Для инженерных приложений это важно, поскольку пульсации АЧХ могут быть ошибочно приняты за реальные частотные особенности сигнала.

Порядок фильтра по умолчанию: 4. Применяется двойная фильтрация (forward-backward, функция `filtfilt`) для обеспечения нулевого фазового сдвига. Это гарантирует, что временные метки событий в сигнале не сдвигаются после фильтрации.

### Оконная функция Ханнинга

При вычислении спектра сигнал конечной длины обрезается, что создаёт эффект утечки спектра: мощность одного пика «растекается» по соседним частотам и может замаскировать соседние пики. Для подавления утечки применяется оконная функция.

Окно Ханнинга (Хан):

```
w(n) = 0.5 · (1 − cos(2π · n / (N − 1))),   n = 0, 1, ..., N−1
```

Оно обеспечивает хороший баланс между разрешением по частоте и подавлением боковых лепестков. Это выбор по умолчанию в приложении. Пользователь может переключиться на окно Хэмминга или прямоугольное окно.

### Метод Уэлча для оценки спектральной плотности мощности

Прямое вычисление FFT от длинной записи даёт шумный спектр с большой дисперсией оценки. Метод Уэлча решает эту проблему: запись разбивается на перекрывающиеся сегменты, для каждого сегмента вычисляется периодограмма с применением оконной функции, затем периодограммы усредняются.

Параметры по умолчанию:
- длина сегмента: 1024 отсчёта;
- перекрытие сегментов: 50%;
- оконная функция: Ханнинга.

При частоте дискретизации 1000 Гц эти параметры дают разрешение по частоте ≈ 0,98 Гц, что достаточно для обнаружения пиков вихревого срыва.

### Поиск пиков в спектре

Пики в спектре ищутся алгоритмом на основе локальных максимумов с порогом относительно медианы спектра. Дополнительно применяется критерий минимальной ширины пика, чтобы отсечь шумовые всплески.

Для каждого найденного пика вычисляется ширина на уровне -3 дБ (то есть на уровне половинной мощности). Узкие высокие пики характерны для гармонических колебаний; широкие пологие пики характерны для широкополосного шума или турбулентных пульсаций.

### RMS-значение

Среднеквадратичное значение (RMS) характеризует энергетику сигнала и связано с интенсивностью вибрации.

```
RMS = sqrt( (1/N) · Σ x²(n) )
```

В приложении RMS вычисляется:
- по всему сигналу после фильтрации;
- в заданной частотной полосе (вычисляется из PSD по теореме Парсеваля);
- в скользящем временном окне (тренд RMS во времени).

## Метрики сравнения эксперимента и расчёта

### Среднеквадратичное отклонение (RMSE)

```
RMSE = sqrt( (1/N) · Σ (y_exp(i) − y_cfd(i))² )
```

Чувствителен к большим отклонениям. Используется для оценки общего качества совпадения.

### Средняя абсолютная погрешность (MAE)

```
MAE = (1/N) · Σ |y_exp(i) − y_cfd(i)|
```

Менее чувствителен к выбросам, чем RMSE. Используется как дополнительный показатель.

### Средняя абсолютная процентная погрешность (MAPE)

```
MAPE = (100/N) · Σ |y_exp(i) − y_cfd(i)| / |y_exp(i)|
```

Показывает относительное отклонение в процентах. Не применяется, когда экспериментальное значение равно или близко к нулю.

### Коэффициент корреляции Пирсона

```
r = Σ (y_exp(i) − ȳ_exp) · (y_cfd(i) − ȳ_cfd) / (N · σ_exp · σ_cfd)
```

Показывает степень линейной зависимости между экспериментальным и расчётным профилями. Значение r = 1 означает идеальное совпадение формы; r = 0 — отсутствие линейной корреляции.

## Классификация риска резонанса

Приложение использует трёхуровневую классификацию:

| Уровень | Условие | Рекомендация |
|---|---|---|
| Безопасный | `|fs − fn| / fn > 0.3` | Режим безопасен при текущих параметрах |
| Наблюдение | `0.1 < |fs − fn| / fn ≤ 0.3` | Контролировать при изменении расхода |
| Критический | `|fs − fn| / fn ≤ 0.1` | Немедленно изменить режим работы |

Дополнительным условием для классификации «критический» является превышение амплитуды пика в спектре вибраций порогового значения, соответствующего нормируемому уровню виброускорения для данного класса оборудования.
