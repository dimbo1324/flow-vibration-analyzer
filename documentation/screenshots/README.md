# Screenshots

The following screenshots should be added manually after running the application.

## How to capture

1. Launch the application: `python main.py`
2. Open an example file from `data/examples/`
3. Run analysis
4. Navigate to each page and take a screenshot

## Required screenshots

| File | Description |
|---|---|
| `main_window.png` | Main application window with dark theme |
| `import_page.png` | Import page with file selection |
| `signal_page.png` | Signal page with time-domain plot |
| `spectrum_page.png` | Spectrum page with PSD and peaks |
| `physics_page.png` | Physics page with risk assessment |
| `report_page.png` | Report page with analysis summary |
| `pdf_report.png` | Generated PDF report preview |
