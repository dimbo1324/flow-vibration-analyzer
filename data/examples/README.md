# IVA Example Data Files

Small demonstration CSV files for the Industrial Vibration Analyzer.
Each file has two columns: `time_s` (seconds) and `signal` (arbitrary units).

| File | Description |
|---|---|
| `example_clean_sine.csv` | Pure 40 Hz sine wave, 6 s at 500 Hz — ideal for verifying spectral peak detection |
| `example_noisy_sine.csv` | 40 Hz sine with Gaussian noise (SNR 15 dB) — typical sensor recording |
| `example_with_gaps.csv` | Clean 40 Hz sine with 10% random NaN gaps — tests gap-handling in the validator |
| `example_risk_like_signal.csv` | Two close frequencies (35 Hz + 38 Hz) representing a near-resonance scenario |

These files are generated by `scripts/generate_synthetic_data.py`.
Re-run the script to regenerate them.
